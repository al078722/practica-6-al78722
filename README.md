# practica-6-al78722
Práctica número 6 - Determinar si un pavimento debe ser rígido o flexible El programa “Determinador del tipo de pavimento” fue creado para ayudar a decidir qué tipo de pavimento conviene construir en una carretera o calle, dependiendo de las condiciones del suelo, el tráfico y el clima del lugar. Este problema es importante en la ingeniería civil, ya que elegir correctamente entre un pavimento rígido o flexible permite que la estructura tenga mayor durabilidad, soporte el tránsito esperado y se reduzcan los costos de mantenimiento.
El código utiliza una interfaz gráfica para que el usuario pueda ingresar los datos necesarios de manera sencilla, sin tener que hacerlo desde la consola. Los tres datos que se solicitan son: el tráfico acumulado (ESAL), el valor del CBR (%) y el tipo de clima. Una vez que se introducen, el programa analiza los valores y determina si el pavimento debe ser flexible o rígido, mostrando el resultado con una breve explicación.
El primer dato es el ESAL (Equivalent Single Axle Load), que significa “Ejes equivalentes de carga”. Representa la cantidad de tráfico pesado (como camiones o autobuses) que circulará por el pavimento durante su vida útil. Si el valor de ESAL es muy alto, indica que habrá mucho tránsito pesado y por lo tanto se recomienda un pavimento rígido, ya que resiste mejor cargas elevadas. Si el valor de ESAL es medio o bajo, el pavimento puede ser flexible, porque el esfuerzo sobre la superficie será menor.
El segundo dato es el CBR (%) (California Bearing Ratio), que es un índice que mide la resistencia del suelo, es decir, qué tan firme o débil es el terreno donde se construirá el pavimento. Si el porcentaje de CBR es bajo (por ejemplo, menor a 5%), significa que el suelo es débil y no soportará bien el peso del tráfico, por lo que se recomienda usar un pavimento rígido. Si el CBR es alto (mayor a 10%), el suelo es fuerte y puede utilizarse un pavimento flexible. El CBR se determina en laboratorio comparando la carga que soporta una muestra de suelo con la que soporta un material estándar.
El tercer dato es el tipo de clima, que también influye en el desempeño del pavimento. En climas húmedos o muy húmedos, el agua puede deteriorar los materiales y reducir la vida útil del pavimento, por lo que se recomiendan estructuras más resistentes. En zonas frías, las variaciones de temperatura pueden causar grietas, mientras que en climas secos o templados las condiciones son más favorables para pavimentos flexibles.
El programa combina estos tres factores mediante condiciones lógicas. Si el tráfico (ESAL) es alto o el suelo (CBR) es débil, el resultado será “Pavimento Rígido”. Si el tráfico es moderado y el suelo tiene buena resistencia, el resultado será “Pavimento Flexible”. Además, el tipo de clima agrega recomendaciones adicionales sobre el cuidado o la calidad de los materiales a emplear.
En resumen, el código busca automatizar de forma sencilla una parte del proceso de diseño vial, utilizando parámetros básicos para que el usuario pueda obtener una recomendación rápida y comprensible sobre el tipo de pavimento más adecuado para una obra determinada.
